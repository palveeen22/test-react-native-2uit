# Тестовое задание для React Native - разработчика 2uit

-- Installation --

1. Clone the repository
    ```bash
    git clone <repository-url>
    cd EcommerceApp
    ```

2. Install dependencies
    ```bash
    npm install
    ```
    ```bash

## Features

- View products
- Search and filter products
- View product details
- Add products to cart
- View cart
- User profile with order history

## Dependencies

- React Native
- React Navigation
- Styled-components

# using npm
npm start

# OR using Yarn
yarn start
```
### For Android

```bash
# using npm
npm run android

# OR using Yarn
yarn android
```

### For iOS

```bash
# using npm
npm run ios

# OR using Yarn
yarn ios
```

Run the app
    ```bash
    npx react-native run-android # For Android
    npx react-native run-ios # For iOS
    ```
